# v4.2.0 Release Notes

> Roadmap milestone only. This work shipped publicly as part of `v4.4.0`, not as a separately published npm release.

## Summary

`v4.2.0` is the trust release. It tightens the public surface, removes misleading contracts, and makes package verification part of the maintained workflow.

## Highlights

- Natural-language parsing docs and types now match the implemented option set.
- The plugin system no longer depends on hidden global registration state.
- Built-package export checks are enforced after build with `npm run test:package`.
- Release hygiene scaffolding was added across the changelog, issue templates, and maintainer docs.

## Verification

- `npm run lint`
- `npm test`
- `npm run build`
- `npm run test:package`

## Notes

- This release is paired with the audit baseline in `docs/releases/v4.2-audit.md`.
